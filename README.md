1、模块有两种工作模式：通过P4处的跳线帽进行选择

    1）跳线帽短接到Lock端(左侧)，表示：门锁控制由J2上的Lock(13)引脚信号控制。

	2）跳线帽短接到P2.0端(右侧)，表示：门锁控制有单片机控制，可通过WiFi连接上位机控制。
	
	3）手机打开电源连接到无线模块HI-Link 718D,使用软件控制开关门
